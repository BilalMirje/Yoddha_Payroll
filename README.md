# Yoddha Payroll

Yoddha Payroll is a backend-based payroll automation system built using Java and Spring Boot.  
The application streamlines employee salary processing, attendance tracking, allowances, and financial record management.

---

## 🚀 Tech Stack

- Java
- Spring Boot
- Spring Security
- MySQL
- REST APIs
- Maven

---

## 📌 Features

- Employee salary processing automation
- Attendance tracking system
- Allowance and advance management
- Vendor payment handling
- Secure authentication and role-based access control
- RESTful API design for scalable integration

---

## 🛠 System Modules

- Employee Management
- Payroll Processing
- Attendance Management
- Allowance & Advance Records
- Vendor Billing

---

## 🔐 Security

- JWT-based authentication
- Role-based authorization
- Secure backend architecture

---

## 📊 Impact

- Reduced manual salary calculation errors
- Improved payroll accuracy and financial transparency
- Increased operational efficiency through backend automation

---
